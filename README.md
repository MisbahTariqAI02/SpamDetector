## How to Run
1. Install [Raylib](https://www.raylib.com/).
2. Open the project in **Dev-C++**.
3. Link the following libraries in your compiler settings:
   `-lraylib -lopengl32 -lgdi32 -lwinmm`
4. Compile and Run (F11).
